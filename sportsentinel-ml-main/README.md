\# SportSentinel



AI-powered sports media protection system.



\## Features



\- Video upload and matching

\- Multimodal fingerprinting using visual, audio, and text signals

\- Violation detection

\- Violation evidence view

\- Anomaly detection

\- Piracy graph/network view

\- DMCA notice generation as PDF
